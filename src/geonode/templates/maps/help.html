{% trans "Using the GeoNode" %}

{% blocktrans %} The CAPRA GeoNode provides access to risk data as well as particular refined data sets which can be freely combined to created maps that visualize particular risks for specific regions. The GeoNode consists of three main sections: {% endblocktrans %}
  • {% trans "A map browser" %} {% blocktrans %} that allows you to expore the maps already contributed by the community. There is also a {% endblocktrans %} {% trans "featured map browser" %} {% blocktrans %} which highlights particularly effective or interesting maps selected by CAPRA. {% endblocktrans %}
  • {% trans "A data browser" %} {% blocktrans %} that allows you to view the individual data sets that are available. You can use these data sets to create your own maps. {% endblocktrans %}
  • {% trans "A map editor" %} {% blocktrans %} that lets you view maps created by others, create your own maps (either from scratch or by customizing existing maps), and share your maps with the CAPRA community. {% endblocktrans %}
